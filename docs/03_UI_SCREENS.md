### مكونات الشاشة الرئيسية:

#### TotalCard
- خلفية متدرجة بلون العملة
- الرقم الكبير: الإجمالي بالعملة
- الرقم الصغير: عدد الرزم الكاملة الإجمالي
- يتحدث فوراً مع كل ضغطة

#### ProfileChips
- شريط أفقي قابل للتمرير
- الشريحة النشطة: مميزة بلون
- زر `[+]` في النهاية لفتح إضافة بروفايل سريعة

#### DenominationGroup (لكل فئة)
```
┌──────────────────────────────────────────────────────┐
│ [اسم الفئة]  [حقل الإدخال]  [مجموع الرزمة]  [-][+] │
│                    ↑                                  │
│            يقبل أرقاماً فقط                          │
│            placeholder: "أوراق"                       │
│            تحت الحقل: "2 رزمة + 50 ورقة"            │
└──────────────────────────────────────────────────────┘
```

**سلوك الحقل:**
- إدخال رقمي فقط
- عند التركيز: يُحدد النص كله
- عند الخروج: يحسب الإجمالي
- تحت الحقل: تلميح `"2 رزمة + 50"` إذا كان الرقم > 100

**زر `+`:**
- يضيف صف جديد تحت الصف الحالي لنفس الفئة
- يظهر فقط في آخر صف من كل مجموعة

**زر `-` (الحذف):**
- يظهر فقط إذا كان هناك أكثر من رزمة في المجموعة
- يحذف الصف الحالي

#### الفئة الطارئة ⚡
- تظهر في آخر القائمة
- علامة `⚡` مميزة
- نفس سلوك الفئات العادية
- لا تُحذف بزر "إعادة ضبط"

#### BottomActions
```
[ + إضافة فئة ]  [ ↺ إعادة ضبط ]  [ 💾 حفظ ]
```

**إضافة فئة** → Dialog يسأل: القيمة + الاسم (اختياري)  
**إعادة ضبط** → تأكيد → يصفر الأرقام (يحتفظ بالفئات الطارئة)  
**حفظ** → Dialog لإضافة ملاحظة → يحفظ في السجل

## التدفق الوظيفي الكامل

```
فتح التطبيق
    │
    ▼
تحميل آخر جلسة محفوظة
    │
    ▼
HomeScreen (البروفايل الأخير نشط)
    │
    ├─── المستخدم يكتب أرقاماً ──→ تحديث فوري للإجمالي + حفظ تلقائي
    │
    ├─── يضغط [+] ──────────────→ إضافة صف رزمة جديدة
    │
    ├─── يغير البروفايل ─────────→ حفظ الجلسة الحالية، تحميل جلسة البروفايل الجديد
    │
    ├─── يضغط [+ إضافة فئة] ────→ Dialog → إضافة فئة طارئة للجلسة
    │
    ├─── يضغط [↺ إعادة ضبط] ───→ تأكيد → تصفير الأرقام
    │
    ├─── يضغط [💾 حفظ] ─────────→ Dialog ملاحظة → حفظ في السجل
    │
    └─── يضغط [📤 مشاركة] ──────→ حفظ تلقائي في السجل + فتح Share Sheet
```

| العنصر | اللون الفاتح | اللون الداكن |

| الرزمة الكاملة | أخضر | أخضر داكن |
| زر الحذف | أحمر | أحمر فاتح |
